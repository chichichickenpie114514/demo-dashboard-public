DATA NOTICE — All data in this directory is SYNTHETIC.

Generated by generate_demo_data.py with random.seed(42):
- 12 months (2025-04 ~ 2026-03)
- 194 JSON files: 16 per-month + 2 root-level
- ~26 fictional facilities (generic Japanese place names)
- ~1,800 synthetic persons (100 common surnames x 40 given names)
- Revenue numbers derived from per-facility service code distributions
- Cross-file consistency: KPI totals = sum of facility billing, person totals = sum of bills

No real company data, customer names, facility names, or financial figures
from the original production dashboard are included.
